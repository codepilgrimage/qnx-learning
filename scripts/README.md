# Scripts

| Script | Purpose |
|---|---|
| `setup_env.sh` | Source the QNX SDP environment and verify `qcc` is available |
| `create_vm.sh` | Create a QEMU/VMware QNX development VM |
| `run_vm.sh` | Boot the dev VM |
| `stop_vm.sh` | Stop the dev VM |
| `clean_vm.sh` | Remove VM disk images / build artifacts |
| `build_examples.sh` | Build every lab and example with a `Makefile` |

All scripts are TODO stubs — fill in the actual VM/toolchain commands for
your environment (QEMU vs VMware, SDP version, target arch).
